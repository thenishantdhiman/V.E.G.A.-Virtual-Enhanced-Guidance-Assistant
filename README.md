**V.E.G.A.** is a Python-based intelligent voice assistant designed to perform real-time automation tasks using **speech recognition** and **AI-powered natural language understanding** through the **Perplexity API**.

### Features
- Voice-activated assistant using **SpeechRecognition**
- AI response generation powered by **OpenAI / Perplexity API**
- Opens websites like Google, YouTube, Gmail, and more via voice
- Can play songs directly through a custom **music library**
- Realistic text-to-speech output using **pyttsx3**
- Modular and extendable architecture

### Tech Stack
- **Programming Language:** Python  
- **Libraries Used:** `speech_recognition`, `pyttsx3`, `webbrowser`, `openai`, `sys`  
- **API:** Perplexity AI Chat API
